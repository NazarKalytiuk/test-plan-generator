# PROJ-1099 — Cancel order endpoint

**Type:** User Story
**Reporter:** Product Owner

## As a / in order to / I want

As a **customer or admin**, I want to **cancel an order** so that the money is not captured.

## Acceptance Criteria

- **AC-1**. The endpoint is `POST /orders/:id/cancel`.

- **AC-3**. Customers can cancel their own orders if the order is pending.
